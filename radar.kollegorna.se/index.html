Crystal

crystal-lang.org

Crystal is a very powerful, static typing language which has the syntax of Ruby. This allows any Ruby developer to write fast, performant, (extremely) type safe code which means much less debugging time, less tests, better quality code and eventually less bugs for the client. I believe that Crystal is the right tool for small APIs that talk to external APIs. Things that you would need rb-dry or Virtus gems in Ruby, Crystal does it out of the box with types.
